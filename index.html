How this works

  1. I trained on a bunch of public sighting reports (like 50k+ after cleaning).
  2. TF-IDF turns the words into numbers, Naive Bayes picks a shape.
  3. This page is a lighter version so you can try it in the browser.
  4. Not proof of UFOs. Its just matching wording to labels from the dataset.

Notebook + slides + writeup are on GitHub if you want the full thing.
