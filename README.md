# UFO Witness Report Analyzer

Live site: https://cdbz8a.github.io/ufo-witness-report-analyzer/

School ML capstone. You paste a UFO witness report and it guesses the shape label (triangle, disk, fireball, light, etc) plus a confidence score and some keywords.

Uses TF-IDF + Multinomial Naive Bayes. Not an LLM.
